### Sigma rule

```

    - https://adsecurity.org/?p=2921

    - attack.t1086

logsource:
    product: windows
    service: powershell-classic
detection:
    selection1:
        EventID: 400
        EngineVersion: 
            - '2.*'
            - '4.*'
            - '5.*'
        HostVersion: '3.*'
    condition: selection1

    - Penetration Tests

```
